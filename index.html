Projects

Live Stock Market Webscraper and Trading Practice Tool. C#

This project involves developing an application that leverages web scraping to collect historical stock data and live prices for select tech stocks. Users can engage in simulated trading within a practice account, where they can buy and sell stocks to build their portfolio using real-time evaluations. The application provides candlestick charts for visualising stock trends and facilitating a virtual trading environment where users can hone their investment skills without risking real money.

AI Models Designed to Predict House Prices From the Ames Housing Dataset. Python

This project involves predicting future house prices using various regression models, such as linear regression, polynomial regression, neural networks, and random forest. Initially, the dataset undergoes preparation steps, including data cleaning, feature selection, and normalisation to ensure its suitability for training the models. The dataset is split into training and testing data to assess their performances accurately and ensure the models generalise well to unseen data and do not overfit. These models are then executed to learn the relationship between house features and prices. Finally, each performance is evaluated using metrics like mean squared error and mean absolute error, providing insights into their predictive accuracy and suitability for forecasting house prices.

Efficiency Focused Route Finding Algorithm For A Complex Data Structure. C++

This project involves building an adjacency list data structure to represent a network of places where each place is linked to its adjacent places using pointers. Using this data structure, various functions are implemented, including breath-first-search for finding the shortest route, route validation to ensure adherence to specific criteria and limitations, and distance calculation between places. This allows efficient traversal of the network and facilitates route planning and analysis within a large geographic area.

Paramount importance is placed on optimising the efficiency of the algorithms within the context of the extensive network. Through meticulous algorithmic design, data structure selection, and code optimization techniques, the project endeavours to minimize computational complexity and resource utilisation while staying 100% accurate.
